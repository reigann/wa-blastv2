# WhatsApp Blaster - Installation Instructions

## QUICK INSTALL (Choose One)

### Option 1: Windows (Fastest)
1. Open this folder in Command Prompt
2. Run: setup.bat
3. Wait for completion
4. Done! Go to "RUNNING" section below

### Option 2: macOS/Linux
1. Open terminal in this folder
2. Run: chmod +x setup.sh && ./setup.sh
3. Wait for completion
4. Done! Go to "RUNNING" section below

### Option 3: Manual Install
Follow COMPLETE_SETUP.md

---

## PREREQUISITES (Must Install First)

- Node.js v18+: https://nodejs.org/
- Python 3.8+: https://www.python.org/
- Chrome Browser: https://www.google.com/chrome/

Verify:
  node --version
  python --version

---

## RUNNING THE PROJECT

After setup completes:

### Terminal 1 - Backend
cd backend
npm start

Expected: "✅ Server running on http://localhost:3001"

### Terminal 2 - Frontend (Open new terminal)
cd frontend
npm run dev

Expected: "➜ Local: http://localhost:5173/"

### Browser
Open: http://localhost:5173
